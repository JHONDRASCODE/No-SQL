Bases de Datos Documentales

Bases de Datos Documentales

Concepto: Almacenan datos en documentos JSON o BSON.

Características: Flexible, escalable horizontalmente, adecuada para datos semiestructurados.

Funcionamiento: Datos almacenados en documentos, cada documento es una unidad autónoma.

Desventajas: Menor consistencia comparada con bases de datos relacionales, complejidad en consultas anidadas.

Página oficial de documentación: MongoDB, Cloud Firestore
